# 프로젝트 개요

## 1. 프로젝트명

Daily Fitness Log

---

## 2. 프로젝트 소개

운동, 식단, 체성분을 하나의 일일기록으로 관리하고,
GPT를 활용하여 운동 및 식단에 대한 피드백과 변화 추이를
확인할 수 있는 개인용 운동 대시보드입니다.

---

## 3. 프로젝트를 시작하게 된 배경


현재 운동은 바디캘린더(모바일 어플)에 기록하고,
식단과 체중은 메모하거나 ChatGPT에 별도로 입력하여
총평을 받고 있다.

기록이 여러 곳에 흩어져 있어 장기적인 변화 추이를
확인하기 어렵고, 매번 ChatGPT에 기록을 복사해서
질문해야 하는 번거로움이 있다.

이를 해결하기 위해 운동, 식단, 체성분을 하나의
일일기록으로 관리하고 ChatGPT를 통해 바로 피드백을
확인할 수 있는 웹 애플리케이션을 개발한다.

---

## 4. 프로젝트 목표

### 서비스 목표

- 운동, 식단, 체성분 기록을 날짜별 일일기록으로 통합 관리한다.
- ChatGPT 분석을 통해 하루 운동 및 식단에 대한 피드백을 쉽게 확인한다.
- 체중, 운동량, 식단 기록의 장기적인 변화 추이를 시각적으로 확인한다.

### 개발 목표

- Spring Security를 활용하여 회원가입 및 로그인 기능을 구현한다.
- JPA를 활용하여 일일기록 중심의 데이터 구조를 설계하고 CRUD를 구현한다.
- React를 활용하여 백엔드 API와 연동되는 화면을 구현한다.
- Git/GitHub를 활용하여 프로젝트를 버전 관리한다.
- Docker를 활용하여 PostgreSQL 실행 환경을 구성한다.

---

## 5. 주요 사용자

### Primary User

- 운동, 식단, 체성분을 꾸준히 기록하고 장기적인 변화 추이를 관리하고 싶은 사용자
- GPT를 활용하여 운동 및 식단에 대한 피드백을 받고 싶은 사용자

### 사용 시나리오

1. 사용자는 일일기록을 생성한다.
2. 운동 기록을 입력하거나 바디캘린더 기록을 붙여넣는다.
3. 하루 식단을 기록한다.
4. 체중 및 체성분을 기록한다.
5. GPT를 통해 운동, 식단 및 하루 총평을 확인한다.
6. 누적된 기록을 기반으로 체중, 운동량, 식단 등의 변화 추이를 확인한다.

## 6. 주요 기능

### 회원
- 회원가입
- 로그인
- 로그아웃

### 일일기록
- 날짜별 일일기록 생성
- 작성중/완료 상태 관리
- 운동, 식단, 체성분, GPT 분석 탭 관리

### 운동기록
- 바디캘린더 운동 기록 텍스트 붙여넣기
- 운동 기록 원문 저장
- 운동 부위, 총 세트 수, 유산소, 걸음 수 등 기록

### 식단기록
- 아침, 점심, 저녁, 간식 기록
- 총 칼로리, 영양성분(탄수화물/단백질/지방) 등 기록

### 체성분
- 체중 기록
- 체성분(골격근량, 체지방량, 체지방률 등) 기록
- 수면, 통증, 컨디션 메모 기록

### GPT 분석
- 오늘 전체 총평 생성
- 운동 분석 생성
- 식단 분석 생성
- 체성분 분석 생성

### 통계
- 체중 변화 그래프
- 주간 운동 횟수
- 주간 총 세트 수
- 평균 칼로리 및 단백질 섭취량 조회
